Cloudflare 它是一整套免费的个人互联网基础设施。域名、网站、CDN、对象存储、数据库、边缘函数、图床、网盘、内网穿透、邮箱转发、验证码、AI 网关，全部能从免费版开始玩。

## 免费 DNS

首先，Cloudflare 本身它就是一个域名服务商。你可以在这里低价的买到很多便宜的域名。买完了以后直接托管在这里。这些域名第二年的续费价格跟第一年基本上都一样。非常的良心。当然，你也可以在 Spaceship 这样的网站买，买完了以后托管在 Cloudflare 上。

域名接进 Cloudflare，你直接拿到一套好用的 DNS 管理系统。

重点是：解析不按查询量收费。不像国内有些大厂特别恶心，竟然按解析次数来收费？吃相不要太难看。

Cloudflare 这就很适合多域名、多子域名、小项目矩阵。

你可以搞：

- `api.xxx.com`
- `img.xxx.com`
- `cdn.xxx.com`
- `docs.xxx.com`
- `status.xxx.com`
- `admin.xxx.com`

就是说，你买了一个域名，你可以托管一万个服务，配置一万个子域名。但是不用花一分钱。不花一分钱。就是这么屌。现在一个域名很便宜，几美元一年。

这些域名你要部署的话，可以开一下官方的小橙云。就是官方帮你代理流量，还能免费帮你配置 HTTPS 的证书，方便的一批。

## 免费 CDN

CDN 缓存是 Cloudflare 最屌的东西。很多人都是通过这个知道 Cloudflare 的。对博客、官网、文档站来说，最直接的价值就是：

访问快一点，源站轻一点，带宽省一点。尤其你用很便宜的 VPS，前面挂 Cloudflare，体感会好很多。

- 个人博客

- 文档站

## 免费 Workers

如果说你的网站想添加一点动态的交互，那就可以用到这个东西，它就类似于网站的后台。是一段部署在 Cloudflare 服务器上的，永久的、免费的、可以运行的、小小的代码。虽然说很小，但是能实现的东西就是五脏俱全，只有你想不到的，没有你做不到的。但是这个代码我们只能用 JavaScript 这样的语言写。你可以用 AI 来帮你写，非常方便。

免费版每天 10 万请求，要啥自行车？如果你一个项目每天能超过 10 万次请求，那你就就可以发财了。直接申请付费版的服务吧。

一个月也只要 5 美元。Worker paid 套餐，可以用到你的公司上市。

很多小项目根本不需要完整后端。一个 Worker 就够了。

## 免费 KV

把它理解为免费版的 Redis。有些东西需要加快地去读取，你就可以放在这个里面。一些小小的配置啊，什么乱七八糟的小的一些值啊，都可以放在里面，往里面塞。它的性能贼好，速度贼快。

## 免费 D1

可以把它理解为免费的 SQLite 数据库。永久地部署在云端。用来存储你项目里的很多表格类型的数据。一个人有 5 个 G 的额度。

## 免费 R2

可以把它理解为免费的网盘。它是兼容亚马逊的 S3 对象存储的。但是比亚马逊可慷慨多了，它的出站网络流量是免费的，不管是多少个 G，它是按次收费，懂吗？按次收费，你就算放一个 10 个 G 的文件，它下载一次，也就是按一次收费。关键是免费用户。它每个月给 1000 万次的下载额度。给 100 万次的上传额度，你说屌不屌？

你可以拿它放：

- 图片
- 附件
- PDF
- 课程资料
- 软件包
- 备份文件
- 用户头像
- Markdown 图片
- 静态资源
- 数据集
- 音频文件
- 小视频素材

## 免费 Email Routing

这个东西就很屌了。你可以自己搭建自己的域名服务器。比如说你有一个域名，然后呢你想自己开一个类似于 QQ 邮箱啊、Foxmail 啊，这样的域名服务。可以帮你收发邮件，免费的，最近刚出的新功能。发邮件的话还是在测试版本，但是也很好用了，我已经用在生产环境了。收邮件、发邮件都能搞。

- 登录

- 评论
- 联系表单
- 等待名单

## 免费 Tunnel

比如说你家里电脑有一个很牛逼的服务。或者说你想搭建什么 Minecraft 服务器啊，什么幻兽帕鲁啊这种乱七八糟的东西，但是呢，你想给远在千里之外的朋友用。那你就装一个这个东西，然后你电脑就会成为一个服务器。千里之外的朋友通过它就可以连上去。

你家里的 NAS、本地开发机、内网服务，可以通过 Cloudflare Tunnel 暴露出来。核心价值：

- 不用公网 IP
- 不用路由器开端口
- 不用暴露源站 IP
- 可以挂自己的域名

比如：

- `nas.xxx.com`
- `dev.xxx.com`
- `panel.xxx.com`
- `n8n.xxx.com`
- `home.xxx.com`

这玩意对家庭服务器玩家太香了。

## 免费 Access

Access 可以给你的应用加身份验证。有了它，你再也不用部署那些傻逼的鉴权系统了。又要登录又要注册，乱七八糟的，还不安全你用它，你直接就是不需要密码，不需要账号。有了它以后就是统一的，你在 Cloudflare 后面板上统一的配置一下你的邮箱，然后你后面就用邮箱验证码登录就行了。你可以配置一个月，比如说你验证一次，一个月以内就不用再次登录了。比如：

- 只有指定邮箱能进
- 只有 Google 登录能进
- 只有 GitHub 登录能进
- 只有团队成员能进

这对保护后台、测试环境、内部工具非常好用。

## 免费 AI Gateway

这个东西就是一个免费的 AI 中转站。你可以把你其他地方搜刮来的一些边角料的供应商的 AI 的 Token 统一的挂在这里，然后你就可以统一的请求这边的接口，就不用在这里找一坨 Key，在那边找一坨 Key，在那边配置一个网址，那边又配置一个网址，统一的就在 Cloudflare 上托管。现在还是全免费的状态，真他妈牛逼。还能帮你做分析呢，比如说哪个供应商延迟多少、缓存多少，都能给你看得清清楚楚的。

它可以帮你看：

- 请求量
- 延迟
- 错误
- 缓存命中
- 模型调用
- 限速
- fallback

你接 OpenAI、Anthropic、Workers AI、各种兼容 API，都可以在前面套一层。

早期做 AI 产品，很适合拿它当统一入口。

适合：
