# 网站图标设计

## 目标

用正式、克制、可长期使用的品牌标记替换人物头像，并保证 16px 浏览器标签尺寸仍然清晰。

## 方案

- 核心符号为几何化的「大」字，直接对应“陈大黄”。
- 深墨蓝圆角方形作为底色，米白色三笔构成字形。
- 不使用人物、渐变、阴影、细线、文字排版或复杂装饰。
- 矢量 SVG 作为母版，再导出 16/32/48/64/180/512px 位图资源。

## 资产

- `site-icon.svg`：矢量母版。
- `avatar-logo.png`：512px 品牌母版，保留旧文件路径兼容现有引用。
- `favicon.png`：64px。
- `favicon.ico`：16/32/48/64px 多尺寸。
- `apple-touch-icon.png`：180px。

## 验证

检查透明边缘、16px 可辨识度、ICO 多尺寸、HTML 缓存版本、生产资源响应和浏览器标签加载。
